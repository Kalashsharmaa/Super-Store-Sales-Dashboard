# Super-Store-Sales-Dashboard
An interactive Power BI dashboard built using the Super Store dataset to analyze sales performance, profit trends, customer segments, shipping modes, payment methods, and regional sales distribution. The dashboard provides actionable insights through dynamic visualizations and KPI tracking to support data-driven business decisions.
📌 Project Overview

This Power BI Sales Dashboard provides a comprehensive analysis of sales performance across different regions, categories, segments, shipping modes, and payment methods. The dashboard is designed to help businesses monitor key performance indicators (KPIs), identify sales trends, and uncover opportunities for growth.

By leveraging interactive visuals and filters, users can explore sales and profit data at multiple levels and make informed business decisions.

🎯 Business Objective

The primary objective of this dashboard is to:

Track overall sales performance.
Monitor profit generation across different regions.
Analyze customer purchasing behavior.
Identify top-performing categories and sub-categories.
Evaluate shipping and payment preferences.
Compare monthly sales and profit trends year-over-year.
